# MCG Consultoria App

Plataforma digital da MCG Consultoria, criada para apoiar a estruturação
comercial e estratégica de empresas do segmento de logística, integrando
método, processos, indicadores e tecnologia.

## Status do Projeto
MVP em desenvolvimento e validação inicial.

## Objetivo do App
Servir como base tecnológica da metodologia MCG, apoiando:
- organização da área comercial
- integração entre áreas da empresa
- acompanhamento de indicadores
- governança de processos
- apoio à tomada de decisão

## Público-Alvo Inicial
- Transportadoras
- Indústrias / Embarcadores
- Operadores Logísticos
- Prestadores de serviços do ecossistema logístico

## Funcionalidades Atuais
- Estrutura inicial do frontend
- Design system definido (UI/UX)
- Base de navegação e layout
- Preparação para módulos de CRM
- Área administrativa (em evolução)

## Funcionalidades Planejadas
- CRM comercial integrado ao método MCG
- Checklists de jornada do cliente
- Calculadora de frete
- Integração com APIs de rotas e pedágio (ex: QualP)
- Integração com meios de pagamento
- Dashboards de indicadores
- Área do cliente e área administrativa MCG
